### Syntax / 函数原型

```C++
int __stdcall 
_path_addrect (
    HPATH hPath,
    float left,
    float top,
    float right,
    float bottom
);
```


### Parameters / 参数

`hPath`

Type: **HPATH**

路径句柄

`left`

Type: **FLOAT**

矩形左边

`top`

Type: **FLOAT**

矩形顶边

`right`

Type: **FLOAT**

矩形右边

`bottom`

Type: **FLOAT**

矩形底边

---

### Return Value / 返回值

Type: INT32

添加矩形
