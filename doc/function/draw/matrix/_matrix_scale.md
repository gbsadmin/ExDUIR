### Syntax / 函数原型

```C++
bool __stdcall 
_matrix_scale (
    int *pMatrix,
    float scaleX,
    float scaleY,
    int order
);
```


### Parameters / 参数

`pMatrix`

Type: **INT32***

矩阵指针

`scaleX`

Type: **FLOAT**

横向缩放倍数

`scaleY`

Type: **FLOAT**

纵向缩放倍数

`order`

Type: **INT32**

顺序

---

### Return Value / 返回值

Type: BOOL

缩放矩阵
