### Syntax / 函数原型

```C++
bool __stdcall 
_layout_table_setinfo (
    int hLayout,
    int aRowHeight,
    int cRows,
    int aCellWidth,
    int cCells
);
```


### Parameters / 参数

`hLayout`

Type: **HANDLE**

布局句柄

`aRowHeight`

Type: **INT32**

行高度

`cRows`

Type: **INT32**

行数量

`aCellWidth`

Type: **INT32**

格宽度

`cCells`

Type: **INT32**

格数量

Type: BOOL
